Siena Galaxy Atlas

This web portal provides dedicated access to the Siena Galaxy Atlas (SGA), a multiwavelength atlas of nearby galaxies selected based on their apparent angular diameter.

For a detailed description of the data, including the data model and the conditions for use, please see the SGA documentation.
